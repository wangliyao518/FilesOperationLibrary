files operation library
==================

Basic files operation package with Robot Framework interface.

Requirements
============

* Python 2.7.x
* setuptools >= 2.1
* pip >= 1.5


installation
=============================

| pip install -U FilesOperationLibrary -i https://pypi.python.org/simple/
